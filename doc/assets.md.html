**Assets✜** Built-in Assets of the [](manual.md.html) Fantasy Console by [Casual Effects](https://casual-effects.com) The console provides thousands of built-in assets so that you can start creating games immediately and make games that have a small size. All assets are Creative Commons licensed and automatically include their license and copyright information in the `CREDITS` constant when loaded. Fonts ==================================================================================== Sounds ==================================================================================== Data ==================================================================================== ## Word Lists `quad://data/english_all_table.json` : All English language word-game words as an object mapping legal words to `true`. This adds about 700 kB to a game download. `quad://data/english_all_array.json` : All English language word-game words as an array of strings. This adds about 700 kB to a game download. `quad://data/english_common_array.json` : The most common ~9000 English language words, based on the New General Service List that has been cleaned. This is a strict subset of `english_all_array.json`. `quad://data/english_common_table.json` : `english_common_array.json` as an object mapping words to `true`. `quad://data/english_sensitive.json` : List of words that a game designer may not want a word game to generate by itself, even if it accepts these words. Words in the English language that may be particularly embarrasing, triggering, sensitive to different groups. Words may appear in both english_offensive and english_sensitive. Represented as an object mapping words in this set to `true`. `quad://data/english_offensive.json` : List of words that a game designer may not want a word game to allow showing or to be communicated between players as they are considered deeply offensive to different groups. Rerpresented as an object mapping words in this set to `true`. assets✜
